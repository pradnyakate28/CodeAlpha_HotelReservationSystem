# Hotel Reservation System
Java Swing project to manage hotel bookings.

## Features:
- Book and cancel rooms
- Room categories: Standard, Deluxe, Suite
- Calculate payment for bookings
- View all booking details in a table
- GUI interface built with Java Swing

## Run Instructions:
1. Open the project in **Eclipse** or **IntelliJ IDEA**.
2. Locate the file `HotelReservationSystemGUI.java`.
3. Run `HotelReservationSystemGUI.java` as a Java Application.
4. The GUI window will appear where you can make or cancel bookings.

## Notes:
- Room rates: Standard = ₹2000, Deluxe = ₹3000, Suite = ₹5000
- Payment is simulated and displayed automatically in the table.
- The project uses Object-Oriented Programming (OOP) principles.
